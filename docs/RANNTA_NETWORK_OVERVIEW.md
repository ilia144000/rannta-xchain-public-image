# RANNTA X-Chain Network Overview

RANNTA X-Chain is a live, independent blockchain network and the native settlement and infrastructure layer of the RANNTA ecosystem.

## Public network role

The network provides:

- Native RNTX settlement and gas accounting
- Public blocks, transactions, accounts, balances, receipts, and state inspection
- JSON-RPC access for compatible wallets and applications
- Explorer-based independent verification
- A native blockchain foundation for RANNTA ecosystem services
- An integration boundary for RANNTA X-Change, RouteX, wallets, and future applications

## Node direction

RANNTA X-Chain follows a controlled operational model. Production node source, validator implementation, consensus internals, signing systems, and deployment automation are maintained privately.

The long-term direction includes broader node participation and movement of suitable ecosystem services toward node-native operation, subject to controlled release, security, and evidence gates.

## Verification rule

Current chain state must be verified using the live RPC and explorer. Static documents provide identity and architecture context, not a substitute for current block, transaction, balance, or health data.

## Repository boundary

This public repository contains only non-sensitive documentation, diagrams, examples, and verification references. It does not expose private implementation logic, validator material, credentials, or production operator configuration.
